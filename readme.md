# Python Flask Tutorial

Following: http://flask.pocoo.org/docs/1.0/tutorial/

To launch virtual env:  
```. venv/bin/activate```

To run:
```bash
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
```
To restart server, just `flask run` again
